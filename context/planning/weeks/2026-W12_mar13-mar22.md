## This Week's Objective

## How This Advances HomeSynapse

These three modules are the absolute foundation of HomeSynapse Core. The EventEnvelope is the universal wire format — every subsystem reads and writes it. The EventBus is the nervous system — every subsystem communicates through it. The Platform API provides the runtime abstraction. Getting these interfaces airtight now means 12 downstream modules can implement against stable contracts without rework. This is the highest-leverage specification work in the entire project.

- Block-level PM deferrals should be flagged when they narrow sprint-plan scope.

### Major Changes & Decisions

1. EventEnvelope is non-generic (DomainEvent marker interface, not `<T>`)
2. CausalContext uses raw Ulid fields (CorrelationId/CausationId/ActorRef wrappers deleted)
3. Ulid + UlidFactory created (binary 128-bit, not in original plan — all typed IDs wrap Ulid)
4. ProcessingMode is CATCHUP/LIVE (subscriber replay), not SYNCHRONOUS/ASYNCHRONOUS
5. String for dynamic-typed payload fields (capability schema not yet defined)
6. Payload types created in Phase 2 (overriding PM Block B Decision 4 deferral)
7. Block C merged into Block B
